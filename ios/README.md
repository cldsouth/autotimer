# iOS Paketleme

Bu klasör, Capacitor `npx cap add ios` komutu çalıştırıldığında native iOS proje dosyalarını içerir.

## İlk Kurulum

1. `npm install`
2. `npx cap add ios`
3. `npx cap sync ios`
4. `npx cap open ios`

> Not: Bu ortamda npm registry erişimi kısıtlı olduğu için `@capacitor/*` paketleri otomatik indirilemedi.

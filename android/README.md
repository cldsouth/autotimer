# Android Paketleme

Bu klasör, Capacitor `npx cap add android` komutu çalıştırıldığında native Android proje dosyalarını içerir.

## İlk Kurulum

1. `npm install`
2. `npx cap add android`
3. `npx cap sync android`
4. `npx cap open android`

> Not: Bu ortamda npm registry erişimi kısıtlı olduğu için `@capacitor/*` paketleri otomatik indirilemedi.
